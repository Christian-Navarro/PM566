Required reading

You are expected to have read these items before class. Most of these linked sections/chapters are quite short.

Interactive visualization

Introduction to plotly: Please read the following sections of Interactive web-based data visualization with R, plotly, and shiny, by Carson Sievert: - Creating Views (Plotly features): Sections 1-3, 5-7 - Publishing Views: Sections 9-12 - Arranging and Animating Views: Sections 13-14 - Linking Multiple Data Views: Sections 15 - Server-side linking with Shiny: Section 17

Introduction to Shiny: - Introduction to Interactive Documents: provides a quick guide to getting started with Shiny.

Introduction to creating dashboards: - Chapter 5: Dashboards (Layout, Components, Shiny) from R Markdown: The Definitive Guide, Yihui Xie, J. J. Allaire, Garrett Grolemund, provides a guide for using Shiny with flexdashboard to create interactive documents from R Markdown

Creating websites

We will be developing a website using rmarkdown’s site generator. Please read: R Markdown: The Definitive Guide, Yihui Xie, J. J. Allaire, Garrett Grolemund - Chapter 10.5: Websites in rmarkdown’s site generator

Optional reading
