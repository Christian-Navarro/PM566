Required reading

Interactive visualization

Introduction to plotly:

Please read the following sections of Interactive web-based data visualization with R, plotly, and shiny, by Carson Sievert:

  • Creating Views (Plotly features): Sections 1-3, 5-7
  • Publishing Views: Sections 9-12
  • Arranging and Animating Views: Sections 13-14
  • Linking Multiple Data Views: Sections 15
  • Server-side linking with Shiny: Section 17

Introduction to Shiny:

  • Introduction to Interactive Documents: provides a quick guide to getting started with Shiny.

Creating websites

We will be developing a website using rmarkdown’s site generator. Please read:

  • Chapter 10.5: Websites in rmarkdown’s site generator in R Markdown: The Definitive Guide, Yihui Xie, J. J. Allaire, Garrett Grolemund

Optional reading

Interactive visualization

  • The official Shiny website includes extensive articles, tutorials, and examples.
  • Coursera course on developing data products, sessions on Shiny, GoogleVis, and Plotly
  • For an introduction to creating dashboards see: Chapter 5: Dashboards (Layout, Components, Shiny) from R Markdown: The Definitive Guide, which provides a guide for using Shiny with flexdashboard to create interactive documents from R Markdown

Website development in rmarkdown

  • For a helpful tutorial on creating a website using rmarkdown, see: Creating websites in R, Emily C. Zabor
  • For more advanced website creation using blogdown and the website generators Hugo and **Jeckyll see Creating websites with R Markdown. Appendix B presents basic information about websites, such as HTML, CSS, and JavaScript.
